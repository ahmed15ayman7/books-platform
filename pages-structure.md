# هيكل صفحات المنصة — Next.js

مرجع لتنفيذ الواجهة: كل صفحة ثنائية اللغة (عربي / إنجليزي) عبر `[locale]` في App Router، بدون تكرار محتوى في هذا الملف.

**مثال مسار:** `/ar/...` و `/en/...`

---

## أنواع الأقسام (مختصر)

| القسم | ماذا يعرض |
|--------|-----------|
| **Hero** | عنوان الصفحة، جملة تعريفية، زر رئيسي |
| **شبكة كتب** | بطاقات كتب: غلاف، عنوان، تصنيف، رابط التفاصيل |
| **تصنيفات كتب** | أيقونات/بطاقات للمجالات (تقنيات، فلسفة، …) |
| **قائمة مقالات** | بطاقات: صورة، عنوان، مقتطف، تاريخ |
| **شبكة ناشرين** | شعار، اسم الدار، الدولة |
| **نص ثابت** | فقرات، عناوين فرعية، قوائم |
| **فيديو** | مشغّل فيديو + عنوان ووصف |
| **نموذج** | حقول إدخال + إرسال |
| **فريق** | صورة، اسم، دور مختصر |
| **تواصل** | هاتف، بريد، روابط |
| **بحث** | حقل بحث + نتائج فورية |
| **فلاتر** | دولة، تصنيف، ترتيب |
| **CTA** | دعوة واضحة لخطوة تالية (انشر، تصفّح، اشترك) |

## الصفحة الرئيسية

**المسار:** `/[locale]`  
**الهدف من الصفحة:** أول انطباع — يوضّح أن المنصة «نافذة العالم على الكتب»: اكتشاف، قراءة، نشر.

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض (Next.js) |
|--------|-----------|-------------|------------------------|
| Hero | شعار المنصة، جملة قيمة («اكتشف كتابك القادم من أي مكان في العالم») | يثبّت الهوية ويوجّه فوراً | `section` بارتفاع ~60vh، خلفية هادئة، زرّان: «تصفّح الكتب» + «انشر كتابك» |
| بحث سريع | حقل بحث عام | يختصر رحلة المستخدم | `SearchCommand` أو شريط بحث sticky؛ اقتراحات: كتب، مقالات، ناشرين |
| تصنيفات كتب | 7 مجالات + كتب مترجمة + مرشحة للترجمة | التصفّح بالاهتمام أسرع من قائمة طويلة | شبكة `CategoryCard` قابلة للسحب أفقي على الموبايل؛ hover يظهر عدد الكتب |
| صدر حديثاً | 8–12 كتاب جديد | يخلق إحساس بالحيوية والجدة | `BookCarousel` أفقي مع autoplay بطيء؛ badge «جديد» |
| حسب التصنيف | مقاطع: تقنيات، دراسات، فلسفة، … (كل واحد 4–6 كتب) | يعمّق الاكتشاف حسب الذوق | `Tabs` أو scroll-spy: عند الوصول للقسم يُحمّل الكتب؛ عنوان القسم + «عرض الكل» |
| انشر كتابك | نص قصير + زر | تحويل زائر إلى مساهم | بانر ملوّن بصرياً مختلف عن باقي الصفحة؛ أيقونة قلم/كتاب |
| دور النشر | شعارات/أسماء ناشرين مميزين | بناء الثقة وربط الكتب بالمصادر | `PublisherMarquee` أو شبكة 12 ناشر؛ فلتر دولة اختياري |
| زبدة الأفكار | 3 مقالات مختصرة | محتوى فكري خفيف يجذب القارئ | بطاقات أفقية كبيرة: صورة + عنوان جذاب + «اقرأ في 5 دقائق» |
| حصاد الكتب | 3 مقالات | نفس الفكرة بصوت «تجميع وقراءة» | تباين بصري عن زبدة الأفكار (لون/أيقونة مختلفة) |
| شاهد كتابك | فيديو أو قصة مميزة واحدة | عاطف + قصة بشرية | `VideoHero` مع poster جذاب؛ نص تحت الفيديو |
| حديث الكتب | 4 عناوين مقالات (بودكاست/نقاش) | محتوى صوتي/مرئي يميّز المنصة | بطاقات بأيقونة تشغيل؛ مدة القراءة/المشاهدة |
| رواية فحكاية | فيديو/قصة أدبية واحدة | جذب محبي الرواية | layout سينمائي: فيديو 16:9 + اقتباس من النص |
| CTA ختامي | «ابدأ رحلتك» — روابط للكتب والمقالات والناشرين | تقليل معدل الخروج | شريط قبل Footer بثلاثة أزرار متساوية |

## كتب — تصنيفات وأرشيف

### كتب مرشحة للترجمة

**المسار:** `/[locale]/books/nominated-for-translation`  
**الهدف:** عرض كتب أجنبية تنتظر ترجمة عربية — جذب مترجمين وناشرين.

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض |
|--------|-----------|-------------|--------------|
| Hero | عنوان + جملة («كتب تستحق أن تصل للقارئ العربي») | توضيح القيمة | عنوان كبير + إحصائية: «+177 كتاب مرشح» |
| فلاتر | لغة أصلية، دولة الناشر، سنة | تضييق النتائج | `Sheet` على الموبايل / شريط أفقي على الديسكتوب |
| شبكة كتب | كل الكتب في التصنيف | المحتوى الأساسي | `BookGrid` responsive؛ infinite scroll أو pagination |
| CTA | «هل تريد ترجمة كتاب؟» → انشر كتابك | تحويل | بانر أسفل الشبكة |

### كتب مترجمة

**المسار:** `/[locale]/books/translated`  
**الهدف:** إبراز ما وصل already للعربية — إلهام وثقة.

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض |
|--------|-----------|-------------|--------------|
| Hero | «مكتبة الترجمات» | تمييز عن المرشحة | خلفية collage لأغلفة مترجمة |
| شبكة كتب | كتب مترجمة | الاكتشاف | Grid + badge «مترجم»؛ ترتيب: الأحدث / الأكثر طلباً |
| ربط | رابط للمرشحة | رحلة مستمرة | «اكتشف كتباً تنتظر الترجمة» |

### تصنيف كتاب (ديناميكي)

**المسار:** `/[locale]/books/category/[slug]`  
**أمثلة:** تقنيات وعلوم، دراسات اجتماعية، فلسفات وثقافات، …

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض |
|--------|-----------|-------------|--------------|
| Hero | اسم التصنيف + وصف سطرين | سياق | Breadcrumb: الرئيسية › كتب › التصنيف |
| شبكة كتب | كتب التصنيف | المحتوى | نفس أرشيف الكتب مع عنوان التصنيف ثابت (sticky subheader) |
| تصنيفات ذات صلة | 3–4 تصنيفات قريبة | اكتشاف إضافي | Chips قابلة للنقر أسفل الشبكة |

### تفاصيل كتاب

**المسار:** `/[locale]/books/[slug]`  
**الهدف:** قرار الشراء/القراءة — كل معلومات الكتاب.

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض |
|--------|-----------|-------------|--------------|
| Hero | غلاف كبير، عنوان AR/EN، تصنيفات | التعريف الفوري | عمودين: غلاف 40% + meta 60% |
| وصف | نبذة، مؤلف، دار نشر، بلد | القراءة العميقة | `Tabs`: نبذة \| تفاصيل \| مراجعات (لاحقاً) |
| كتب مشابهة | 4–6 من نفس التصنيف | الإبقاء على الموقع | Carousel أفقي أسفل الصفحة |
| CTA | رابط الناشر، مشاركة، حفظ | تفاعل | أزرار: مشاركة، «المزيد من [التصنيف]» |

**الهدف:** مقالات تلخّص وتستعرض كتباً — قارئ يريد فكرة سريعة قبل الشراء.

| القسم | ماذا يعرض | لماذا نعرضه | مقترح العرض |
|--------|-----------|-------------|--------------|

| **Skeleton loaders** | إحساس سرعة أثناء جلب البيانات |
| **Empty states** | رسالة ودية عند عدم وجود نتائج + اقتراح بديل |

## تنقل مقترح (Header)

```
الرئيسية | كتب ▾ | مقالات ▾ | ناشرون | انشر كتابك | من نحن
```

**كتب ▾:** كل التصنيفات، مترجمة، مرشحة للترجمة  
**مقالات ▾:** حصاد الكتب، زبدة الأفكار، حديث الكتب، شاهد كتابك، رواية فحكاية، العالم يقرأ

## ملاحظات تنفيذ Next.js

- استخدم `next-intl` أو `next-i18n` لـ `[locale]` وترجمة الواجهة.
- الصفحات ذات القوائم (كتب، مقالات، ناشرين): Server Components + pagination.
- الصفحات الثابتة (من نحن، خصوصية): MDX أو CMS لاحقاً؛ مبدئياً محتوى في ملفات `content/`.
- «حديث الكتب» و«شاهد كتابك»: اربط كل بطاقة بمقال حقيقي من API بدل نصوص ثابتة مكررة.
- صفحات «بودكاست» و«فيديوهات» المنفصلة في الموقع القديم: ادمجها كأقسام داخل «حديث الكتب» أو «شاهد كتابك» لتجنب صفحات فارغة.
